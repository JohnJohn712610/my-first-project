## Улучшения landing.html — по итогам исследования

### 🔴 Высокий приоритет (влияет на конверсию)
- [ ] **Блок «Боли родителей»** — 3–4 узнаваемые ситуации («ребёнок зубрит правила, но на уроке молчит», «сменили двух репетиторов — результата нет»). Emotional hook — родитель узнаёт себя и остаётся. Нужны реальные фразы от Светланы.
- [ ] **Тарифы / пакеты** — сейчас только «2000₽ / занятие». Добавить пакеты (4 / 8 / 16 уроков), выделить «популярный», добавить кнопку выбора. Нужны реальные цены от Светланы.
- [ ] **«Как записаться» в 3 шага** — блок снимающий барьер: «Напишите → Светлана ответит за 2 часа → Первый урок бесплатно». Сейчас непонятно, что будет после клика на Telegram.

### 🟡 Средний приоритет (усиливает доверие)
- [ ] **Анимированные счётчики** — «120+ учеников», «8 лет опыта», «97% улучшают оценку». Считаются при скролле до блока. Нужны реальные цифры от Светланы.
- [ ] **Sticky-кнопка на мобильном** — кнопка «Записаться» всегда видна внизу экрана при прокрутке. 70% трафика с телефона.

### 🟢 Низкий приоритет (дизайн и полировка)
- [ ] SVG-волны между секциями вместо прямых горизонтальных границ
- [ ] Шрифт Nunito вместо Segoe UI — более дружелюбный для детской аудитории
- [ ] Переменные шрифты: заголовок «нарастает» по жирности при загрузке
