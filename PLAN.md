## Лендинг репетитора — что нужно от тебя (по brief.md)

> Это контент и данные, которые нужно собрать **до вёрстки**. Без них лендинг будет с заглушками.

### 📸 Фото и визуал

- [ ] **Живое фото Светланы** — не постановочное, в процессе урока или за ноутбуком. Формат: JPG/PNG, минимум 800×800px. Нужно для: герой-секции и блока «Преподаватель»
- [ ] **Фото или аватары для отзывов** — реальные фото родителей (или инициалы + цвет, если фото нет)
- [ ] **Скриншоты из Telegram** — благодарности от родителей в переписке. Обрезать лишнее, оставить имя и текст

### 📊 Цифры для счётчиков

- [ ] **Сколько учеников занимались** — хотя бы приблизительно (пример: «80+»)
- [ ] **Сколько лет опыта работы с детьми**
- [ ] **Средний балл ОГЭ/ЕГЭ учеников** (если есть данные)
- [ ] **Из скольких городов ученики** (если онлайн — можно посчитать)

### 💰 Цены и тарифы

- [ ] **Цена индивидуального урока** (45 мин) — конкретная сумма или «от X ₽»
- [ ] **Цена занятия в мини-группе** (2–4 ученика, 60 мин)
- [ ] **Цена подготовки к ОГЭ/ЕГЭ** — отдельный тариф или такой же как индивидуальный
- [ ] **Что входит в бесплатный пробный урок** — 45 мин? Диагностика? Что скажешь родителю?

### 🎓 Данные о преподавателе

- [ ] **Образование** — вуз, специальность, год окончания
- [ ] **Сертификаты и квалификации** — Cambridge, CELTA, курсы — всё что есть
- [ ] **Специализации** — с какого возраста берёшь детей (сейчас: от 7 лет, младшая группа 7–9 лет добавлена), что умеешь лучше всего
- [ ] **Личная цитата** — 2–3 предложения от первого лица для секции «Преподаватель». Пример: «Я работаю с детьми N лет и знаю, как объяснить Present Perfect без слёз»

### 💬 Отзывы (минимум 3)

Для каждого отзыва нужно:
- [ ] **Отзыв 1** — имя родителя, город, возраст ребёнка, запрос (что хотели), результат (что получили)
- [ ] **Отзыв 2** — то же самое
- [ ] **Отзыв 3** — то же самое

> Формат результата: не «всё понравилось», а конкретно — «За 4 месяца с A1 до A2» или «Сдала ОГЭ на 4»

### ❓ FAQ — ответы на вопросы

- [ ] **Как оплачивать?** — Сбербанк, ЮMoney, карта? Помесячно или пакетом?
- [ ] **Можно ли перенести урок?** — за сколько часов предупредить?
- [ ] **Что если не понравится?** — условия возврата
- [ ] **С какого возраста берёшь?** — минимальный возраст (сейчас: от 7 лет)

## Лендинг-портфолио цифрового мастера — новый проект

> Отдельный лендинг: сайты + приложения + ИИ-агенты. Анализ в research.md → Раздел D.

### 🔴 Подготовка (нужны данные — сделать до вёрстки)
- [ ] **Написать оффер** — формула: [результат] для [кто] за [срок/условие]. Пример: «Запускаю сайт за 7 дней — с оплатой после приёмки». Не стек технологий, а результат для клиента
- [ ] **Собрать 3–5 кейсов** — для каждого: задача клиента → что сделано → результат в цифрах + живая ссылка на проект
- [ ] **Определить цены / диапазоны** — хотя бы «от X» для каждой услуги: лендинг / приложение / ИИ-агент. Без цен 70% уходят
- [ ] **Собрать 3 отзыва** — с именем, должностью, компанией клиента и конкретным результатом (не «всё понравилось»)
- [ ] **Сделать живое фото** — не логотип, не иконка. Клиент покупает человека, которому доверяет

### 🔴 Структура страницы (8 блоков — в этом порядке)
- [ ] **Герой** — оффер результатом + кому + CTA «Обсудить проект» + время ответа «отвечу за 2 часа»
- [ ] **Услуги** — 3 направления (сайты / приложения / ИИ-агенты) с кратким описанием и ценой/диапазоном
- [ ] **Кейсы** — 3–5 проектов: задача → решение → результат + живая ссылка
- [ ] **О себе** — фото + 3–4 предложения от первого лица + 3–4 цифры (проектов, лет, клиентов)
- [ ] **Отзывы** — 3 отзыва с именем, должностью, компанией, конкретным результатом
- [ ] **Как работаем** — 4 шага от заявки до готового проекта
- [ ] **FAQ** — 5 вопросов закрывающих основные возражения («сколько стоит», «как долго», «что если не понравится»)
- [ ] **Финальный CTA** — «Обсудить проект» + Telegram + «Отвечу в течение 2 часов»

### 🟡 UX и конверсия
- [ ] **Форма — максимум 2 поля**: имя + Telegram. Всё остальное на созвоне
- [ ] **Sticky-кнопка на мобильном** — «Обсудить проект» всегда видна внизу экрана
- [ ] **5-секундный тест** — открыть на мобильном и проверить: понятен ли заголовок, видно ли фото, есть ли цена, понятен ли CTA
- [ ] **Скорость загрузки** — скриншоты кейсов сжать до <200 КБ, проверить в PageSpeed

### 🟢 Доверие и полировка
- [ ] Логотипы компаний-клиентов (хотя бы 3–5)
- [ ] Ссылка на GitHub с активными репозиториями
- [ ] Сертификаты (OpenAI, Google Cloud, AWS — если есть)
- [ ] Анимированные счётчики: «17 проектов», «средний срок 12 дней»
