## Лендинг-портфолио цифрового мастера — новый проект

> Отдельный лендинг: сайты + приложения + ИИ-агенты. Анализ в research.md → Раздел D.

### 🔴 Подготовка (нужны данные — сделать до вёрстки)
- [ ] **Написать оффер** — формула: [результат] для [кто] за [срок/условие]. Пример: «Запускаю сайт за 7 дней — с оплатой после приёмки». Не стек технологий, а результат для клиента
- [ ] **Собрать 3–5 кейсов** — для каждого: задача клиента → что сделано → результат в цифрах + живая ссылка на проект
- [ ] **Определить цены / диапазоны** — хотя бы «от X» для каждой услуги: лендинг / приложение / ИИ-агент. Без цен 70% уходят
- [ ] **Собрать 3 отзыва** — с именем, должностью, компанией клиента и конкретным результатом (не «всё понравилось»)
- [ ] **Сделать живое фото** — не логотип, не иконка. Клиент покупает человека, которому доверяет

### 🔴 Структура страницы (8 блоков — в этом порядке)
- [ ] **Герой** — оффер результатом + кому + CTA «Обсудить проект» + время ответа «отвечу за 2 часа»
- [ ] **Услуги** — 3 направления (сайты / приложения / ИИ-агенты) с кратким описанием и ценой/диапазоном
- [ ] **Кейсы** — 3–5 проектов: задача → решение → результат + живая ссылка
- [ ] **О себе** — фото + 3–4 предложения от первого лица + 3–4 цифры (проектов, лет, клиентов)
- [ ] **Отзывы** — 3 отзыва с именем, должностью, компанией, конкретным результатом
- [ ] **Как работаем** — 4 шага от заявки до готового проекта
- [ ] **FAQ** — 5 вопросов закрывающих основные возражения («сколько стоит», «как долго», «что если не понравится»)
- [ ] **Финальный CTA** — «Обсудить проект» + Telegram + «Отвечу в течение 2 часов»

### 🟡 UX и конверсия
- [ ] **Форма — максимум 2 поля**: имя + Telegram. Всё остальное на созвоне
- [ ] **Sticky-кнопка на мобильном** — «Обсудить проект» всегда видна внизу экрана
- [ ] **5-секундный тест** — открыть на мобильном и проверить: понятен ли заголовок, видно ли фото, есть ли цена, понятен ли CTA
- [ ] **Скорость загрузки** — скриншоты кейсов сжать до <200 КБ, проверить в PageSpeed

### 🟢 Доверие и полировка
- [ ] Логотипы компаний-клиентов (хотя бы 3–5)
- [ ] Ссылка на GitHub с активными репозиториями
- [ ] Сертификаты (OpenAI, Google Cloud, AWS — если есть)
- [ ] Анимированные счётчики: «17 проектов», «средний срок 12 дней»
