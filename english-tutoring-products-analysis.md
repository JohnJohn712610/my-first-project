## 1. AI English Conversation Partner (Telegram Bot)

**Что делает:** Бот в Telegram симулирует живой разговор на английском, исправляет ошибки и объясняет правила.

**Аналоги:** Duolingo Bot, SnapChat My AI, десятки indie-ботов на Product Hunt

**Ключевой функционал:**
- Пользователь пишет по-английски → бот отвечает, выделяет ошибки, объясняет правило
- Режимы: разговор, проверка грамматики, перевод с объяснением
- Уровни сложности (A1–C1), топики для разговора
- Ежедневная "разминка" — 5 фраз для перевода

**Путь пользователя:**
```
/start → выбор уровня → выбор топика → чат →
бот исправляет → пользователь повторяет → streak-счётчик
```

**Монетизация:**
- Freemium: 10 сообщений/день бесплатно
- Подписка: 299–499 ₽/мес за безлимит
- Разовые пакеты уроков: 990 ₽ за 30 дней

**Сложность:** 🟢 Просто

**Как сделать через Claude Code:**
- Python + `python-telegram-bot` + Anthropic API
- Claude Code генерирует весь код за 1-2 часа
- Деплой на Railway.app или Render (бесплатный тир)
- MVP за **1 день**

**Как адаптировать под себя:**
- Добавить свой голос/персонажа ("Джеймс из Лондона")
- Специализация: бизнес-английский, IELTS-подготовка, разговорный
- Целевая аудитория: студенты, IT-специалисты, мамы в декрете

## 2. AI Grammar & Writing Coach (веб-приложение)

**Что делает:** Пользователь вставляет текст на английском → получает детальный разбор ошибок с объяснениями на русском.

**Аналоги:** Grammarly (но дорогой и без русских объяснений), LanguageTool

**Ключевой функционал:**
- Текстовое поле + кнопка "Проверить"
- Цветная подсветка ошибок (красный = грубая, жёлтый = стиль)
- Каждая ошибка → правило + пример правильного использования на русском
- История проверок, прогресс (топ-5 твоих ошибок)
- "Rewrite" — Claude переписывает текст в правильном стиле

**Путь пользователя:**
```
Вставить текст → нажать "Проверить" →
увидеть ошибки → кликнуть на ошибку →
прочитать объяснение → исправить → сохранить
```

**Монетизация:**
- Freemium: 3 проверки/день бесплатно
- Pro: 499 ₽/мес — безлимит + история + экспорт PDF
- B2B: лицензии для школ и языковых центров

**Сложность:** 🟢 Просто

**Как сделать через Lovable:**
- Промпт в Lovable: "Create a React app for English grammar checking with Claude API"
- Lovable собирает UI за 20 минут
- Подключить Anthropic API через Supabase Edge Functions
- MVP за **1 день**

**Как адаптировать:**
- Добавить специализированные режимы: деловой email, IELTS-эссе, резюме
- Интеграция с Google Docs через расширение Chrome
- Версия для преподавателей — проверять работы учеников пачкой

## 3. Flashcard Vocabulary Builder с AI-контекстом

**Что делает:** Умные карточки для запоминания слов — AI создаёт примеры предложений, объясняет нюансы, тестирует через интервальное повторение.

**Аналоги:** Anki (сложный), Quizlet (дорогой), Memrise

**Ключевой функционал:**
- Добавить слово → AI автоматически создаёт: перевод, 3 примера, синонимы, коллокации
- Алгоритм спейсд-репетишн (показывает карточки в нужный момент)
- Дневная цель: X слов в день, streak-система
- Импорт из текста/PDF — вытащить все незнакомые слова
- Тесты: выбор из 4-х вариантов, написать слово, составить предложение

**Путь пользователя:**
```
Добавить слово/импорт текста → AI создаёт карточку →
ежедневные повторения → тест →
статистика запомненных слов
```

**Монетизация:**
- Freemium: до 100 слов бесплатно
- Premium: 399 ₽/мес — безлимит + AI-генерация + импорт
- Готовые наборы слов (топ-3000, бизнес, IELTS) — разовая покупка

**Сложность:** 🟡 Средне

**Как сделать:**
- Lovable/Cursor для UI + Supabase для базы данных
- Алгоритм SM-2 (спейсд-репетишн) — 50 строк кода, Claude Code напишет
- MVP без алгоритма повторений (просто карточки): **1 день**
- Полная версия с повторениями: **3 дня**

**Как адаптировать:**
- Тематические наборы под свою аудиторию (медицина, IT, юриспруденция)
- Совместные наборы для классов/групп
- Интеграция с YouTube — изучать слова из видео

## 4. English Tutor Marketplace (мини-Preply)

**Что делает:** Платформа где ученики находят репетиторов по английскому, бронируют уроки и платят онлайн.

**Аналоги:** Preply, italki, Repetitor.ru

**Ключевой функционал:**
- Профили репетиторов: фото, видео-визитка, специализация, цена, расписание
- Поиск/фильтры: цена, уровень, специализация, отзывы
- Бронирование урока через встроенный календарь
- Оплата (ЮКасса / Stripe) — деньги хранятся до урока
- После урока: отзыв + рейтинг
- AI-помощник: подобрать репетитора по запросу

**Путь пользователя:**
```
Ученик: поиск → профиль → бронирование → оплата → урок → отзыв
Репетитор: регистрация → профиль → подтверждение урока → выплата
```

**Монетизация:**
- Комиссия 15–20% с каждого урока
- Платные продвижения в поиске для репетиторов
- Подписка для репетиторов: 990 ₽/мес = 0% комиссии

**Сложность:** 🔴 Сложно

**Как сделать:**
- Lovable для UI + Supabase для данных + Stripe/ЮКасса для платежей
- Самая сложная часть — платёжная логика и escrow
- Реалистичный MVP (без платежей, просто профили + бронирование): **3-5 дней**
- С платежами: **2-3 недели**

**Как адаптировать:**
- Узкая ниша: только репетиторы для IELTS/TOEFL, только для детей, только онлайн
- Корпоративная версия: компании покупают уроки для сотрудников
- Добавить AI-тестирование уровня при регистрации

## 5. AI IELTS/TOEFL Writing Coach

**Что делает:** Пользователь пишет эссе для IELTS → AI проверяет по официальным критериям, ставит примерный балл и даёт детальный фидбек.

**Аналоги:** IELTS.org practice tools, Magoosh, платные репетиторы

**Ключевой функционал:**
- Задание из банка реальных IELTS тем (500+ тем)
- Таймер (40 минут как на реальном экзамене)
- После отправки: оценка по 4 критериям (Task Achievement, Coherence, Lexical Resource, Grammar)
- Примерный балл IELTS (6.0 / 6.5 / 7.0)
- Конкретные советы: "Используй больше linking words", "Эта фраза слишком разговорная"
- История эссе + динамика прогресса
- Образцовые эссе на 8.0+ для сравнения

**Путь пользователя:**
```
Выбрать тему → таймер → написать → получить оценку →
сравнить с образцом → повторить через 3 дня
```

**Монетизация:**
- Freemium: 2 проверки/день бесплатно
- Pro: 799 ₽/мес — безлимит + история + PDF-отчёты
- Пакет "Подготовка к IELTS": 2990 ₽ за 3 месяца

**Сложность:** 🟢 Просто

**Как сделать через Claude Code:**
- Flask/FastAPI бэкенд + простой React фронт
- Промпт для Claude с официальными критериями IELTS — ключевая часть
- Supabase для хранения эссе и истории
- MVP: **1-2 дня** (без истории и платежей)

**Как адаптировать:**
- Добавить Speaking Coach: запись голоса → транскрипция → анализ
- Специализация на Speaking Part 2 (монолог)
- Версия для TOEFL Writing, Cambridge CAE

## 6. English Lesson Planner для репетиторов (B2B SaaS)

**Что делает:** AI помогает репетиторам создавать планы уроков, упражнения, домашние задания и материалы за минуты, а не часы.

**Аналоги:** Нет прямого аналога на русском рынке

**Ключевой функционал:**
- Форма: уровень студента + тема + цель урока → AI генерирует план
- Генерация упражнений: fill in the gaps, matching, multiple choice
- Библиотека готовых шаблонов уроков
- Экспорт в PDF/Google Docs
- Трекер студентов: прогресс, заметки после урока, домашние задания

**Путь пользователя:**
```
Репетитор: новый урок → заполнить форму →
AI генерирует план → отредактировать →
экспортировать → после урока заметки
```

**Монетизация:**
- Trial: 7 дней бесплатно
- Pro: 899 ₽/мес (до 20 студентов)
- Business: 2490 ₽/мес (безлимит + командный доступ)

**Сложность:** 🟡 Средне

**Как сделать:**
- Lovable для UI, Supabase для данных, Claude API для генерации
- Самое сложное — красивый PDF-экспорт (использовать react-pdf)
- MVP (только генерация планов без трекера): **2 дня**

**Как адаптировать:**
- Расширить до других предметов (математика, история)
- Интеграция с Zoom/Google Meet — автоматические заметки с урока
- Marketplace шаблонов — репетиторы продают свои материалы

## 7. Daily English Challenge Bot (n8n автоматизация)

**Что делает:** Бот в Telegram/WhatsApp каждый день отправляет подписчикам задание на английском и проверяет ответ.

**Аналоги:** Duolingo Daily, Lingodeer

**Ключевой функционал:**
- Утром: слово дня + задание (составить предложение / перевести / выбрать форму)
- Пользователь отвечает → AI проверяет → даёт фидбек
- Streak-система: 7 дней подряд = badge
- Еженедельный дайджест: "Ты выучил 14 слов, твой прогресс..."
- Платный канал: углублённые задания, голосовые упражнения

**Путь пользователя:**
```
Подписка → ежедневное задание в 9:00 →
ответ → фидбек → стрик → недельный отчёт
```

**Монетизация:**
- Бесплатный базовый канал + платный Premium: 199 ₽/мес
- Корпоративные подписки для команд
- Партнёрства с языковыми школами

**Сложность:** 🟢 Просто

**Как сделать через n8n:**
- n8n: триггер по расписанию → Claude API → Telegram Bot API
- Supabase для хранения прогресса пользователей
- Вся логика в n8n без единой строчки кода
- MVP: **4-8 часов**

**Как адаптировать:**
- Тематические каналы: Business English Daily, IELTS Daily Challenge
- Групповые челленджи: класс соревнуется друг с другом
- Реферальная программа: пригласи друга = +7 дней Premium
