# Legaltech Product Ideas

Анализ IT-продуктов в нише юриспруденции, пригодных для вайб-кодинга.
Составлен: 2026-05-15

---

## 1. AI Contract Reviewer (Spellbook / Robin AI)
Загружаешь договор — получаешь анализ рисков, красные флаги и предложения по правкам.

- **Функционал:** загрузка PDF/DOCX, выбор типа анализа, структурированный отчёт, экспорт
- **Монетизация:** Freemium — 3 анализа бесплатно, далее $29/мес или $5 за документ
- **Сложность:** 🟢 просто
- **Стек:** Flask + Claude API + PyMuPDF
- **MVP:** 1 день (основа уже есть в contract-checker/)
- **Адаптация:** специализация под тип договора (поставка, аренда, трудовые)

---

## 2. Legal Document Generator (Clerky / Juro)
Заполняешь форму-опросник — на выходе готовый юридический документ.

- **Функционал:** выбор типа документа, заполнение формы, генерация, скачать Word/PDF
- **Монетизация:** Pay-per-doc $3-10 или подписка $49/мес
- **Сложность:** 🟢 просто
- **Стек:** Lovable (UI) + Claude API + python-docx + n8n
- **MVP:** 1-2 дня
- **Адаптация:** библиотека под отрасль (производство, строительство, торговля)

---

## 3. Legal Q&A Chatbot (DoNotPay упрощённый)
Чат-бот отвечает на юридические вопросы простым языком.

- **Функционал:** чат, пошаговые ответы, предложение шаблона или консультации
- **Монетизация:** B2B лицензия юрфирмам $199/мес или лидген для юристов
- **Сложность:** 🟢 просто
- **Стек:** Telegram Bot API + Claude API + n8n
- **MVP:** 4-6 часов
- **Адаптация:** специализация по нише (трудовое, корпоративное, защита потребителей)

---

## 4. Contract Clause Library + Search (Clause.ai)
База готовых юридических формулировок с семантическим поиском.

- **Функционал:** поиск по типу условия, варианты от мягкого до жёсткого, AI-адаптация
- **Монетизация:** подписка $19-49/мес для юристов
- **Сложность:** 🟡 средне
- **Стек:** Supabase pgvector + Claude embeddings + Lovable
- **MVP:** 2-3 дня
- **Адаптация:** фокус на конкретный тип договоров (поставка, госконтракты, IT)

---

## 5. Court Document Summarizer (Casetext CoCounsel)
Загружаешь судебное решение — получаешь краткое резюме + ключевые факты.

- **Функционал:** загрузка PDF, выбор формата резюме, экспорт
- **Монетизация:** $0.50-2 за документ или подписка $99/мес
- **Сложность:** 🟡 средне
- **Стек:** PyMuPDF + чанкинг + Claude API (200k контекст)
- **MVP:** 2 дня
- **Адаптация:** специализация под арбитражные решения, КС, апелляционные суды

---

## 6. Legal Due Diligence Automation (Luminance)
Автоматически проверяет пакет документов на риски и несоответствия.

- **Функционал:** загрузка папки файлов, AI анализирует каждый, сводный отчёт рисков
- **Монетизация:** Enterprise $500+/мес или per-project $200-1000
- **Сложность:** 🔴 сложно
- **Стек:** Claude API batch + n8n оркестрация
- **MVP:** 3-5 дней (упрощённая версия)
- **Адаптация:** фокус на конкретный тип сделок (покупка бизнеса, лицензии, проверка контрагентов)

---

## 7. Lawyer Marketplace + AI Matching (Avvo / Rocket Lawyer)
Клиент описывает проблему — AI подбирает юриста и организует консультацию.

- **Функционал:** описание проблемы, AI-классификация, подбор юристов, оплата + видео
- **Монетизация:** комиссия 15-20% с консультации
- **Сложность:** 🔴 сложно
- **Стек:** Lovable + n8n + Stripe
- **MVP:** 3-4 дня (без AI-матчинга)
- **Адаптация:** специализация под регион или нишу права
