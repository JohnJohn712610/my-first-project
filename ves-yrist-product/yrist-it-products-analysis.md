# Анализ IT-продуктов для правовой экспертизы договоров

---

## Продукт 1 — AI Contract Reviewer
**Аналог: Spellbook, Harvey AI**
Загружаешь договор — получаешь структурированный правовой анализ с рисками и рекомендациями.

**Ключевой функционал:**
- Загрузка PDF/DOCX договора
- AI анализирует: предмет, ответственность, сроки, форс-мажор, подсудность
- Выдаёт: список рисков по категориям (высокий / средний / низкий), конкретные формулировки для замены
- Экспорт результата в PDF / Word

**Путь пользователя:** загрузил → подождал 30 сек → получил заключение → скачал

**Монетизация:** Freemium — 3 договора бесплатно, далее 990 руб./мес. или 199 руб./за анализ

**Сложность:** 🟡 средне

**Как сделать через Claude Code:**
```
Stack: Flask/FastAPI + Claude API + PyPDF2 + простой HTML-фронт
Время на MVP: 2-3 дня
```
- День 1: загрузка файла + парсинг текста + промпт для анализа
- День 2: красивый вывод результата + экспорт
- День 3: лендинг + оплата через ЮKassa

**Адаптация под себя:** специализируй под конкретный тип договора (поставка, аренда, подряд) — сделай промпт под договоры вашей компании, добавь внутренние стандарты.

## Продукт 2 — Telegram-бот «Юрист в кармане»
**Аналог: DoNotPay (упрощённая версия)**
Бот принимает фото/текст договора, отвечает на вопросы о нём в режиме диалога.

**Ключевой функционал:**
- /start → выбор типа документа
- Загрузка PDF или вставка текста
- Диалог: "Есть ли ответственность за просрочку?" → конкретный ответ со ссылкой на пункт
- /заключение → итоговый документ
- История запросов

**Путь пользователя:** написал боту → загрузил договор → задаёт вопросы → получил готовое заключение

**Монетизация:** подписка 490 руб./мес. через Telegram Stars или ЮKassa

**Сложность:** 🟢 просто

**Как сделать через n8n + Claude:**
```
n8n workflow: Telegram Trigger → Extract Text → Claude API → Send Message
Время на MVP: 1 день
```
- n8n: Telegram Bot node → Claude node → ответ
- Без единой строки кода
- Хранение истории: Google Sheets или Airtable

**Адаптация под себя:** настрой системный промпт под специфику производственных договоров, добавь базу типовых рисков вашей отрасли.

## Продукт 3 — Генератор правового заключения
**Аналог: Lexion, нет прямого аналога на RU-рынке — ниша свободна**
Юрист вводит параметры договора → система генерирует готовое заключение в нужном корпоративном формате.

**Ключевой функционал:**
- Форма: тип договора, стороны, предмет, спорные пункты (с подсказками)
- Выбор шаблона заключения (внутренний стандарт)
- Генерация структурированного документа: введение / анализ / выводы / рекомендации
- Редактор с возможностью правки прямо в браузере
- Скачать Word / PDF

**Путь пользователя:** заполнил форму (5 мин) → получил заключение → поправил → скачал

**Монетизация:** B2B лицензия — от 5 000 руб./мес. на юридический отдел

**Сложность:** 🟡 средне

**Как сделать через Lovable:**
```
Lovable prompt: "Создай веб-приложение: форма ввода параметров договора,
интеграция с Claude API, генерация Word-документа, авторизация"
Время на MVP: 2-3 дня
```
- Lovable генерирует фронт + React
- Claude API через edge functions
- Экспорт через docx библиотеку

**Адаптация под себя:** загрузи корпоративный шаблон заключения, обучи промпт на примерах ваших реальных заключений.

## Продукт 4 — Конструктор протокола разногласий
**Аналог: нет прямого — уникальная ниша**
Загружаешь договор контрагента → система предлагает редакцию спорных пунктов → собирает готовый протокол разногласий.

**Ключевой функционал:**
- Загрузка договора контрагента
- AI выделяет пункты, требующие корректировки
- По каждому пункту: «редакция контрагента» vs «наша редакция» (можно выбрать из предложенных или написать своё)
- Автосборка протокола разногласий в формате таблицы
- Скачать готовый документ

**Путь пользователя:** загрузил → выбрал спорные пункты → утвердил формулировки → скачал протокол

**Монетизация:** Pay-per-use 299 руб./протокол или подписка 1 990 руб./мес.

**Сложность:** 🔴 сложно (из-за табличного формата вывода и Word-генерации)

**Как сделать через Claude Code:**
```
Stack: Python + Claude API + python-docx + Streamlit (для быстрого UI)
Время на MVP: 3-4 дня
```
- Streamlit даёт UI без фронтенд-разработки
- python-docx собирает таблицу протокола
- Самое сложное — промпт для правильного разбора пунктов

**Адаптация под себя:** добавь библиотеку «наших стандартных формулировок» — тогда система будет автоматически подставлять ваши типовые условия.

## Продукт 5 — Трекер договоров с AI-мониторингом
**Аналог: Ironclad (упрощённо), ContractPodAi**
CRM для договоров: все договоры в одном месте, AI напоминает о сроках, флагует риски.

**Ключевой функционал:**
- База договоров (загрузить/добавить вручную)
- AI извлекает ключевые даты: срок действия, оплаты, продления
- Напоминания в Telegram/email
- Дашборд: статус договоров, приближающиеся дедлайны
- Поиск по базе: «найди все договоры с контрагентом X»

**Путь пользователя:** загрузил договор → AI заполнил карточку → получаешь напоминания → всё под контролем

**Монетизация:** SaaS 2 990 руб./мес. за команду (до 10 пользователей)

**Сложность:** 🔴 сложно (база данных + уведомления + мультипользователь)

**Как сделать через Lovable + Supabase:**
```
Lovable: фронт + авторизация
Supabase: база данных + storage для файлов
n8n: напоминания по расписанию
Время на MVP: 4-5 дней
```

**Адаптация под себя:** добавь кастомные поля для вашей отрасли (тип поставки, завод-получатель, ответственный юрист).

## Продукт 6 — AI-ассистент для анализа правовых актов
**Аналог: ChatPDF, но специализированный**
Загружаешь локальный НПА, положение, приказ — задаёшь вопросы и получаешь ответы с цитатами.

**Ключевой функционал:**
- Загрузка документа (PDF/DOCX)
- Чат с документом: "Какие обязанности возложены на юридический отдел?"
- Ответы с точными цитатами и номерами пунктов
- Сохранение сессии, история вопросов
- Возможность загрузить несколько документов для сравнения

**Путь пользователя:** загрузил НПА → задал вопрос → получил ответ с цитатой → нашёл нужный пункт за 10 сек вместо 30 мин

**Монетизация:** внутренний корпоративный инструмент (без внешней монетизации) или B2B 1 490 руб./мес./пользователь

**Сложность:** 🟢 просто

**Как сделать через Claude Code:**
```
Stack: Python + Claude API + PyPDF2 + Streamlit
Время на MVP: 1-2 дня
```
Это буквально 50 строк кода — Streamlit + file_uploader + Claude с текстом в контексте.

**Адаптация под себя:** добавь корпоративную базу НПА, настрой под специфику производственного предприятия.

## Продукт 7 — Маркетплейс юридических шаблонов с AI-заполнением
**Аналог: Genie AI (UK), частично — Консультант+**
Библиотека проверенных шаблонов договоров → AI заполняет под конкретную сделку по данным пользователя.

**Ключевой функционал:**
- Каталог шаблонов (договор поставки, аренды, подряда, доп. соглашения)
- Мастер: вопрос-ответ (стороны, предмет, условия) → AI вставляет данные в шаблон
- Предпросмотр и редактирование
- Скачать Word
- Возможность загрузить свой шаблон

**Путь пользователя:** выбрал шаблон → ответил на 10 вопросов → скачал заполненный договор

**Монетизация:** freemium (3 шаблона бесплатно) + подписка 790 руб./мес.

**Сложность:** 🟡 средне

**Как сделать через Lovable:**
```
Lovable: каталог + форма-мастер + авторизация
Claude API: заполнение шаблона
python-docx / docxtpl: вставка данных в Word
Время на MVP: 2-3 дня
```

**Адаптация под себя:** загрузи корпоративные шаблоны вашей компании — получишь внутренний инструмент стандартизации документов.
