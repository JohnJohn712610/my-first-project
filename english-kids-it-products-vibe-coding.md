## 1. Parent Progress Bot (по модели ClassDojo)

**Что делает:** Бот автоматически отправляет родителям отчёт после каждого урока — что прошли, что задано, над чем работать.

**Аналог:** ClassDojo (50 млн пользователей), Seesaw — но они сложные и на английском. Простой Telegram-бот закрывает ту же потребность.

**Ключевой функционал:**
- Репетитор вводит в бот: имя ученика + краткий итог урока
- Бот форматирует и отправляет родителю готовый отчёт
- Родитель видит: тему урока, домашнее задание, похвалу/замечание
- Еженедельный дайджест: прогресс за неделю
- Напоминания родителю за час до урока

**Путь пользователя:**
```
Репетитор после урока:
→ Пишет боту: "Маша, Present Simple, д/з стр.45"
→ Бот красиво оформляет и отправляет маме Маши

Родитель:
→ Получает отчёт в Telegram
→ Видит д/з → контролирует ребёнка
→ Доверяет репетитору больше
```

**Монетизация:**
- Продаётся как дополнительная услуга: +500–800 ₽/мес к цене урока
- Или как аргумент для повышения ценника на 1000–2000 ₽/мес
- Лицензия другим репетиторам: 990 ₽/мес

**Сложность:** 🟢 Просто

**Как сделать через n8n:**
- Telegram Bot через @BotFather (5 минут)
- n8n workflow: Telegram Trigger → форматирование текста → отправка родителю
- Supabase: таблица учеников (имя → Telegram родителя)
- MVP за **1 день**

**Как адаптировать:**
- Добавить фото работы ребёнка прямо в отчёт
- Автоматические мотивационные стикеры детям ("Молодец! 🌟")
- Продать как готовый инструмент другим репетиторам

## 2. Квиз "Узнай уровень английского ребёнка" (по модели Placement Test)

**Что делает:** Родитель проходит квиз за 3 минуты и получает точный уровень ребёнка + рекомендации.

**Аналог:** British Council Placement Test, Cambridge Online Test — официальные, скучные, без воронки продаж. Твой квиз ведёт к записи на урок.

**Ключевой функционал:**
- 10–12 вопросов: родитель отвечает про ребёнка (класс, оценки, может ли читать/писать)
- + 5 тестовых заданий для ребёнка (вставить слово, выбрать перевод)
- На выходе: уровень A1/A2/B1 + "что это значит для вашего ребёнка"
- Автоматически: рекомендация записаться на диагностику к тебе
- Email/Telegram с результатом → ты получаешь контакт

**Путь пользователя:**
```
Мама видит рекламу/пост →
→ Проходит квиз (3 мин) →
→ Получает результат "уровень A2, это значит..." →
→ Оффер: "Запишитесь на бесплатный урок-диагностику" →
→ Оставляет номер телефона
```

**Монетизация:**
- Лид-магнит: бесплатно → конверсия в платные уроки
- Продажа квиза другим репетиторам: 1990 ₽ (белый лейбл)
- Партнёрство с языковыми центрами

**Сложность:** 🟢 Просто

**Как сделать через Tally.so:**
- Tally.so — бесплатный конструктор квизов с логикой
- Настроить условные переходы: разные ветки для разных классов
- Интеграция с Telegram через n8n — лиды приходят прямо тебе
- MVP за **4–6 часов**

**Как адаптировать:**
- Отдельные квизы: "Готов ли ребёнок к ОГЭ", "Нужен ли репетитор"
- Виральность: "Поделитесь результатом в родительском чате"
- A/B тест разных офферов в конце

## 3. AI Помощник по домашнему заданию для детей

**Что делает:** Ребёнок фотографирует или вводит задание по английскому → AI объясняет решение по-русски, не просто даёт ответ.

**Аналог:** Photomath (но для математики, 220 млн скачиваний), Socratic by Google — нет хорошего аналога для английского на русском языке.

**Ключевой функционал:**
- Ввод задания текстом или фото (OCR)
- AI объясняет правило простым языком (уровень 10-летнего)
- Не даёт готовый ответ сразу — сначала объяснение, потом подсказка
- Режимы: грамматика, перевод, чтение, топики
- Родительский режим: история запросов ребёнка

**Путь пользователя:**
```
Ребёнок (10 лет):
→ "Не понимаю Present Perfect"
→ Вводит упражнение из учебника
→ Получает: правило + пример + объяснение на русском
→ Пробует сам → проверяет ответ

Родитель:
→ Видит что ребёнок занимался 20 минут
→ Доверяет продукту
```

**Монетизация:**
- Freemium: 5 объяснений/день бесплатно
- Подписка: 299 ₽/мес — безлимит
- Семейный тариф: 499 ₽/мес (до 3 детей)

**Сложность:** 🟡 Средне

**Как сделать через Lovable:**
- Lovable собирает UI: чат-интерфейс для детей (яркий, простой)
- Anthropic API с системным промптом: "Ты учитель английского для детей 7-16 лет. Объясняй правила простым языком по-русски. Никогда не давай готовый ответ сразу."
- Supabase Auth для родительских аккаунтов
- MVP (без фото, только текст): **2 дня**
- С OCR (фото заданий): **4–5 дней**

**Как адаптировать:**
- Брендировать под себя — твой аватар отвечает детям
- Интегрировать с твоими уроками: бот знает программу твоих учеников
- Продать доступ ученикам как бонус к занятиям

## 4. ОГЭ/ЕГЭ Трекер подготовки (по модели Magoosh)

**Что делает:** Веб-приложение для отслеживания подготовки к ОГЭ/ЕГЭ по английскому — что сдано, что осталось, прогноз балла.

**Аналог:** Magoosh ($250/мес, только английский рынок), Яндекс.Репетитор — нет персонального трекера с прогнозом.

**Ключевой функционал:**
- Структура экзамена разбита по разделам (аудирование, чтение, письмо, говорение)
- Ученик отмечает что отработал → прогресс-бар
- AI анализирует слабые места → даёт план на неделю
- Прогноз балла на основе текущего прогресса
- Репетитор видит дашборд всех учеников

**Путь пользователя:**
```
Ученик 9 класса:
→ Регистрация → выбор экзамена (ОГЭ/ЕГЭ) + дата
→ Вводит что уже знает
→ Получает план: "До экзамена 4 месяца, нужно 3 ч/нед"
→ Каждую неделю отмечает прогресс
→ Видит "Прогноз: 78 баллов"

Репетитор:
→ Открывает дашборд → видит прогресс всех учеников
→ Понимает кому нужно больше внимания
```

**Монетизация:**
- Для учеников: бесплатно (лид-магнит для привлечения к репетитору)
- Для репетиторов: 990 ₽/мес — дашборд всех учеников
- White-label для школ: 9 990 ₽/мес

**Сложность:** 🟡 Средне

**Как сделать через Lovable:**
- Lovable: UI с прогресс-барами, чекбоксами по темам ОГЭ/ЕГЭ
- Supabase: хранить прогресс каждого ученика
- Claude API: генерировать персональный план на неделю
- MVP (только трекер без AI-плана): **2 дня**
- С AI-планом: **3–4 дня**

**Как адаптировать:**
- Добавить банк заданий ОГЭ/ЕГЭ с разбором ошибок
- Интеграция с Google Calendar — план занятий
- Родительский доступ: мама видит прогресс ребёнка

## 5. Мини-маркетплейс репетиторов английского для детей

**Что делает:** Платформа где родители находят проверенных репетиторов по английскому специально для детей, с фильтрами по возрасту и целям.

**Аналог:** Preply, Profi.ru — но без специализации на детях. Нет площадки "только детские репетиторы с опытом работы с детьми".

**Ключевой функционал:**
- Профили репетиторов: опыт с детьми, возраст учеников, методики
- Фильтры: возраст ребёнка, цель (ОГЭ / школьная программа / разговорный), цена
- AI-подбор: родитель описывает ребёнка → система предлагает 3 репетитора
- Отзывы от родителей (не учеников)
- Пробный урок со скидкой 50%

**Путь пользователя:**
```
Мама:
→ "Нужен репетитор для дочери 11 лет, готовимся к ОГЭ"
→ AI подбирает 3 варианта
→ Смотрит профиль, читает отзывы родителей
→ Записывается на пробный урок

Репетитор:
→ Создаёт профиль → проходит верификацию
→ Получает заявки от родителей
→ Платит комиссию 15% с урока
```

**Монетизация:**
- Комиссия 15–20% с каждого урока
- Топ-размещение в поиске: 1990 ₽/мес
- Верификация/бейдж "Проверенный педагог": 990 ₽ разово

**Сложность:** 🔴 Сложно

**Как сделать:**
- Lovable для UI + Supabase для данных и авторизации
- Без платежей (просто контакты): MVP за **4–5 дней**
- С платежами (Stripe/ЮКасса): **3–4 недели**
- Начать как агрегатор без транзакций — свести репетитора и родителя, монетизировать позже

**Как адаптировать:**
- Запустить сначала как Telegram-канал с карточками репетиторов
- Узкая ниша: только Москва / только онлайн / только ОГЭ-ЕГЭ
- Ты сам — первый репетитор на платформе

## 6. Автоматизация записи и напоминаний для репетитора

**Что делает:** Система которая сама записывает новых учеников, напоминает о занятиях и собирает оплату — без участия репетитора.

**Аналог:** Calendly + Stripe + Zapier — но нет готового русскоязычного решения для репетиторов.

**Ключевой функционал:**
- Страница записи: родитель выбирает время из доступных слотов
- Автоматическое подтверждение в Telegram/email
- Напоминание за 24 часа и за 1 час до урока
- После урока: автоматический счёт на оплату
- Если не оплатил за 3 дня — напоминание
- Статистика: загрузка, доход, отмены

**Путь пользователя:**
```
Родитель находит ссылку на запись →
→ Выбирает свободный слот →
→ Получает подтверждение в Telegram →
→ За час до урока: напоминание →
→ После урока: ссылка на оплату →
→ Оплатил → подтверждение
```

**Монетизация:**
- Личное использование: экономит 2–3 часа в неделю на организации
- Продажа шаблона другим репетиторам: 2990 ₽ (n8n workflow)
- SaaS для репетиторов: 599 ₽/мес

**Сложность:** 🟡 Средне

**Как сделать через n8n:**
- Calendly (бесплатный тир) для страницы записи
- n8n: триггер при новой записи → Telegram-уведомление
- n8n: Schedule Trigger → напоминания за 24ч и 1ч
- ЮКасса или Telegram Stars для оплаты
- MVP (запись + напоминания, без оплаты): **1 день**
- С оплатой: **2–3 дня**

**Как адаптировать:**
- Добавить Google Calendar синхронизацию
- Автоматический опрос после пробного урока
- CRM: карточка каждого ученика с историей уроков

## 7. Игровые карточки-флешкарды для детей (по модели Quizlet)

**Что делает:** Веб-приложение с яркими карточками для запоминания английских слов — ребёнок учит слова в игровой форме, родитель видит прогресс.

**Аналог:** Quizlet (60 млн пользователей), Anki — но сложные и не адаптированы для детей и российской школьной программы.

**Ключевой функционал:**
- Готовые наборы по темам школьной программы (Spotlight, Rainbow English, Happy English)
- Карточка: картинка + слово + произношение + перевод
- Мини-игры: угадай слово, собери пару, напиши по памяти
- Звёздочки и награды за прохождение (геймификация)
- Репетитор создаёт набор → отправляет ученику ссылку

**Путь пользователя:**
```
Репетитор после урока:
→ Создаёт набор "Unit 5 — Animals" из 15 слов
→ Отправляет ссылку ученику

Ребёнок дома:
→ Открывает на телефоне
→ Учит карточки (5 мин)
→ Играет в мини-игру
→ Получает звёздочки

Родитель:
→ Видит что ребёнок занимался 8 минут
```

**Монетизация:**
- Бесплатно для учеников (удержание аудитории репетитора)
- Репетиторский аккаунт: 490 ₽/мес — создавать наборы без ограничений
- Готовые наборы по учебникам: 199 ₽ за учебник

**Сложность:** 🟡 Средне

**Как сделать через Lovable:**
- Lovable: яркий детский UI с карточками и анимациями
- Anthropic API: автоматически генерировать примеры предложений для слов
- Supabase: хранить наборы и прогресс
- MVP (карточки без игр): **2 дня**
- С мини-играми: **4–5 дней**

**Как адаптировать:**
- Загрузить все слова из популярных российских учебников
- Совместный режим: весь класс учит одни слова, соревнование
- Интеграция с ботом отчётности — родитель видит сколько слов выучил
