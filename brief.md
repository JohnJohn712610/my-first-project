## 1. Цель сайта

Одностраничный продающий лендинг с **единственной целью** — чтобы родитель оставил заявку на бесплатный пробный урок для ребёнка.

Никаких вторичных целей: не подписка на рассылку, не переход в блог. Один экран — одна мысль — одно действие.

**KPI лендинга:** конверсия в заявку ≥ 3–5%.

## 4. Стиль дизайна

### Цветовая схема — Dark Premium

Из research.md, Раздел C: схема 1 «Dark Premium» — дорого, современно, энергично.

```
Фон страницы:   #111111
Фон карточек:   #1e1e1e
Обводка:        #2a2a2a
Текст основной: #ffffff
Текст вторичный:#a0a0a0
Акцент:         #FF6B2B  ← оранжевый (кнопки, заголовки, иконки)
Акцент hover:   #e55a1f
Зелёный успех:  #4CAF50  ← для меток «правильно», бейджей
```

### Шрифты

Подключаем через `@import` из Google Fonts (CDN — допустимо, не библиотека):

```
Заголовки H1–H3: Nunito (variable, weight 600–800)
  — округлый, дружелюбный, читается на мобильном
  — идеален для детской аудитории (LinguaTrip, LingoGalaxy)

Текст body:      Inter (variable, weight 400–500)
  — стандарт читабельности
```

**Правила типографики:**
- `font-size: clamp(1rem, 2.5vw, 1.25rem)` — fluid type, масштабируется без media query
- `letter-spacing: -0.02em` для больших заголовков
- `line-height: 1.6` для body
- Максимум 2 шрифта на странице (строго)

### Настроение и визуальный язык

- **Дорого и современно** — тёмный фон, аккуратные карточки с `border-radius: 20px`
- **Живой и человечный** — реальное фото Светланы, текст от первого лица, без канцелярита
- **Детский, но не наивный** — оранжевый акцент тёплый, без пастели и мультяшек
- **Структурный** — много воздуха между блоками, каждая секция — одна мысль
- **SVG-волны** между секциями для плавных переходов (без резких границ)

## 5. Обязательные элементы

### Кнопки

**Основная CTA:**
```css
background: #FF6B2B;
color: #fff;
border-radius: 50px;
padding: 18px 40px;
font-size: 1.1rem;
font-weight: 700;
animation: pulse 2.5s infinite;  /* привлекает взгляд */
```

Текст кнопки: **«Записаться на бесплатный урок»** — везде одинаково. Не менять формулировку.

CTA появляется 3 раза: герой → секция цен → финальный блок.

**Вторичная кнопка** (если нужна): outlined, оранжевая обводка, прозрачный фон.

### Форма захвата

**Максимум 2 поля** (правило из research.md — иначе 80% закрывают):

```html
[ Ваше имя        ]
[ Телефон / Telegram ]
[ Записаться на бесплатный урок → ]
```

Под кнопкой мелким текстом: «Отвечаю в течение 2 часов. Данные не передаю третьим лицам.»

Форма появляется в модальном окне при нажатии на любую CTA-кнопку (один компонент на всю страницу).

### Анимации (реализация без библиотек, чистый CSS + JS)

| Элемент | Анимация | Реализация |
|---|---|---|
| Кнопка-CTA | Pulse-эффект | CSS @keyframes pulse |
| Карточки | Fade-in снизу при скролле | IntersectionObserver + CSS transition |
| Счётчики (цифры) | Считаются от 0 до цели | JS setInterval + IntersectionObserver |
| Карточки при hover | Lift-эффект (поднятие + тень) | CSS transform + box-shadow |
| Ссылки | Animated underline | CSS ::after pseudo-element |
| Аккордеон FAQ | Плавное раскрытие | CSS max-height transition |
| Модальная форма | Fade + scale | CSS opacity + transform |

**Запрещено:** автоплей видео со звуком, анимации текста по одной букве, всплывающие окна через 3 секунды.

### Мобильная версия — обязательные элементы

- **Sticky-кнопка** внизу экрана: «Записаться бесплатно» — всегда видна при скролле на мобильном
- Кнопки: минимум `height: 52px`, `padding: 16px 24px` — достаточно для нажатия пальцем
- Шрифт H1 на мобильном: не меньше 28px
- Нет горизонтального скролла при любой ширине от 320px
- Навигация на мобильном: только кнопка «Записаться», без гамбургера (лендинг — не сайт)

## 6. Технические требования

- **Один HTML-файл** — CSS в `<style>`, JS в `<script>`. Никаких отдельных файлов.
- **Никаких внешних библиотек** — без Bootstrap, jQuery, Swiper, AOS и т.п.
- Google Fonts — подключать через `@import url(...)` в `<style>` (CDN допустим).
- Семантическая разметка: `<header>`, `<main>`, `<section>`, `<footer>`, `<nav>`.
- Якорные ссылки в навигации: `href="#for-whom"`, `href="#prices"` и т.д.
- `<meta name="viewport" content="width=device-width, initial-scale=1">` — обязательно.
- Картинки — `loading="lazy"`, `alt` на русском.
- Время загрузки: изображения оптимизировать, не больше 200 КБ каждое.

### Из экспертных оценок (Раздел D)

- Заголовок H1 — **результат для клиента**, не описание деятельности (Эксперт 1 + 2)
- CTA повторяется **3 раза** по странице (Эксперт 1)
- Форма — **максимум 2 поля** (Эксперт 1)
- **Открытые цены** — без «стоимость по запросу» (Эксперт 2 + 3)
- **Чёткий следующий шаг** после заявки — «напишу в Telegram за 2 часа» (Эксперт 2)
- **Живое фото** Светланы, имя, текст от первого лица (Эксперт 2 + 3)
- Блок **«Боли родителей»** — emotional hook, сейчас нет (Раздел C, приоритеты)

### Из трендов дизайна (Раздел C)

- **Bold-типографика** в герое — огромный заголовок, акцент на ключевом слове
- **Scroll-triggered анимации** — карточки въезжают, счётчики считаются
- **Trust-first дизайн** — реальные цифры, фото учителя, скриншоты результатов
- **Mobile-native** — sticky-кнопка внизу экрана на мобильном
- **Минимализм + 1 CTA на секцию** — много воздуха, одна мысль на блок
- **Органические формы** — SVG-волны между секциями
